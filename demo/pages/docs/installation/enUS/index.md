# Installation

> Please note that naive-ui only support Vue3. If you are using Vue2, you may look at other libraries.

Use npm to install.

```bash
npm i -D naive-ui
```

## Fonts

```bash
npm i -D vfonts
```

## Icons

naive-ui recommends using [xicons](https://www.xicons.org) as icon library.
