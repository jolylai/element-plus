# Progress 进度条

展示操作的当前进度。

## 何时使用

在操作需要较长时间才能完成时，为用户显示该操作的当前进度和状态。

- 当一个操作会打断当前界面，或者需要在后台运行，且耗时可能超过 2 秒时；
- 当需要显示一个操作完成的百分比时。

## Progress

| 参数    | 说明                                   | 类型   | 默认值 |
| ------- | -------------------------------------- | ------ | ------ |
| name    | 文件名                                 | string | -      |
| percent | 上传进度                               | number | -      |
| status  | 上传状态，不同状态展示颜色也会有所不同 |        | -      |
| uid     | 唯一标识符，不设置时会自动生成         | string | -      |
| url     | 下载地址                               | string | -      |
